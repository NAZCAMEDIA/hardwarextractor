# HardwareXtractor

Completa fichas técnicas de equipos informáticos en minutos, no horas. Solo introduces el modelo de cada componente (CPU, placa base, RAM, GPU y disco) y la app rellena la ficha con especificaciones verificadas y su fuente.

## Para qué sirve
- Acelera la documentación técnica de equipos.
- Reduce errores manuales y mantiene trazabilidad por campo.
- Exporta fichas listas para inventario o integraciones.

## Descarga (macOS)
Carpeta estándar de descargas:
- `downloads/HardwareXtractor.dmg`
- `downloads/HardwareXtractor.app`

## Uso básico
1) Abre la app.
2) Escribe un componente (ej. “Intel Core i7‑12700K”, “ASUS B550‑F”, “Samsung 970 EVO Plus”).
3) Repite con los demás componentes.
4) Exporta la ficha a CSV.

## Importante
- No se inventan datos: si no hay fuente confiable, verás UNKNOWN/NA.
- Cada valor incluye su origen (URL y tipo de fuente).
- Las fuentes oficiales tienen prioridad sobre referencias.
