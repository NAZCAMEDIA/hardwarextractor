# hxtractor (CLI)

CLI interactivo para analizar componentes, añadirlos a una ficha técnica y exportar a CSV/MD.

## Uso
```bash
npx hxtractor
```

Requiere `python3` instalado (Estrategia B).

Si es la primera vez, instala dependencias Python:
```bash
python3 -m pip install scrapy parsel requests lxml
```
